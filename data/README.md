# Data manifests

- `DATA_VERSIONS.txt` — pinned versions for all annotation/reference files used end-to-end.
- `MODEL_REVISIONS.txt` — pinned model revisions and SHAs for Evo 2 7B and cross-architecture baselines.
- `external/README.md` — download URLs and approximate sizes for external data files (not committed).

External BED/FA/VCF/NPY/parquet data files are not committed. See `external/README.md` for sources.
