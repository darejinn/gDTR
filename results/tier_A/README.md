# Tier-A results

These tables back the headline numbers cited in Sections 4-5 of the paper.

| File | What it is |
|---|---|
| `A1_restriction_of_range.csv` | Per-context (intergenic / intron / exon / UTRs / splice) c-mean, H-mean, H-std, H-iqr, and ρ. Self-consistent c from the master forward. |
| `A2_promoter_elements.csv` | Per promoter-element class (TSS / TATA / CAAT / GC-box) Cohen's d vs intron and vs GC-matched control. |
| `A3_effect_sizes.csv` | ε² and rank-biserial unified effect-size summary. |
| `A3_kw_omnibus.json` | Kruskal-Wallis omnibus on settling depth across contexts. |
| `A4_fig2_baseline.csv` | Pooled chr22 intron baseline + per-context Cohen's d (Fig 2 sanity). |
| `A1_optionA.csv` / `A2_optionA.csv` / `A4_optionA.csv` | OptionA variant: paper-canonical c joined with freshly computed H_t. |
| `fig2_unified_d_summary.json` | Pooled chr17+chr22 intron baseline plus splice donor/acceptor and ENCODE cCRE-ELS Cohen's d (Fig 2 panel (b)). |
